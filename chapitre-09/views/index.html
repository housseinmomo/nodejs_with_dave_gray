Routes et methodes HTTP utiliser pour exploiter l'API mugen sama

  • HTTP Mehtod : [GET] | Path: /employees : Get all Employees
  • HTTP Mehtod : [GET] | Path: /employees/id : Get one Employee by id
  • HTTP Mehtod : [POST] | Path: /employees | body: {provide employee} : Create New Employee
  • HTTP Mehtod : [PUT] | Path: /employees/id : Update Employee with id
  • HTTP Mehtod : [Delete] | Path: /employees/ | body: {provide id employee} : Delete Employee with id
